# CLAUDE.md

본 문서는 Claude(LLM) 가 본 레포지토리에서 작업할 때 따라야 할 행동 규칙을 명시한다.

## 커밋 정책

- `git commit` 을 직접 실행하지 않는다. 모든 커밋은 개발자가 직접 수행한다.
- 개발자가 커밋 메시지 제안을 요청하면, 변경 내용에 맞는 메시지를 제안한다.
- 메시지 형식: 1행에 적절한 접두사(`feat`, `fix`, `refactor`, `test`, `docs`, `chore` 등) + 변경 요약을 작성하고, 상세 내용은 본문에 하이픈(`-`) 목록으로 간단히 정리한다.
